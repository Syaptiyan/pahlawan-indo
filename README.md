# 🇮🇩 Pahlawan Indonesia

Daftar lengkap Pahlawan Nasional Indonesia dengan fitur pencarian, filter, dan detail informasi.

## 🚀 Live Demo

👉 [https://syaptiyan.github.io/pahlawan-indo](https://syaptiyan.github.io/pahlawan-indo)

## ✨ Fitur

- **170+ Pahlawan Nasional** — data lengkap dari API
- **Pencarian** — cari berdasarkan nama atau deskripsi
- **Filter per Daerah** — Aceh, Sumatera, Jawa, Bali, Kalimantan, Sulawesi, Maluku, Papua
- **Badge Kategori** — Pahlawan Revolusi, Kemerdekaan, Perintis, Pendidikan, Tokoh Agama, Pembangunan
- **Tag Otomatis** — Militer, Politisi, Aktivis, Ulama, Sultan
- **Detail Modal** — info lengkap (lahir, wafat, usia, tahun penetapan)
- **Dark Theme** — desain modern dan elegan
- **Responsive** — tampil sempurna di mobile dan desktop

## 🛠️ Teknologi

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- JavaScript
- [Alpine.js](https://alpinejs.dev/) — reactive framework
- [Font Awesome](https://fontawesome.com/) — icons
- API Pahlawan Indonesia
